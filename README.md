## Guide
MAX_PATCH_VALIDITY_RULES.md contains all rules for creating a "legal" and functioning Max patch.


PATCH_LAYOUT_GUIDE.md contains all rules regarding patch layout and visual organization

examples directory contains all valid, functioning Max Patches and its corresponding MaxPy python files.

These can be used as a reference for generating valid MaxPy python files and Max patches using LLMs/coding agents of your choice.
